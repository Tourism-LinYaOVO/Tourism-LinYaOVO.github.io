响应式导航栏 - 优化移动端过渡

这是一个简洁、美观、易用的响应式导航栏。在电脑端显示完整菜单，移动端显示汉堡菜单。已优化移动端分类菜单的展开/收起动画。

优化内容：

  • 修复了移动端分类菜单收起时的卡顿问题
  • 使用height替代max-height，实现更平滑的过渡效果
  • 添加了下拉菜单项的渐入动画
  • 使用cubic-bezier缓动函数优化动画曲线
  • 优化了交互体验，点击分类菜单不再关闭整个菜单栏

响应式导航栏功能说明

此导航栏具有以下特点：

响应式设计

  • 电脑端显示完整菜单
  • 移动端显示汉堡菜单
  • 平滑的过渡动画
  • 自适应各种屏幕尺寸

用户友好

  • 简洁美观的视觉设计
  • 清晰的信息层次
  • 直观的交互反馈
  • 易于触摸操作（移动端）

功能完整

  • 支持分类下拉菜单
  • 活动状态高亮显示
  • 支持键盘导航
  • 代码结构清晰

移动端交互说明：

1. 点击汉堡按钮展开/收起主菜单

2. 点击带有下拉箭头的分类菜单（如"产品"、"服务"）可以展开/收起子菜单

3. 子菜单项有平滑的渐入动画，收起时无卡顿

4. 点击子菜单项会跳转到相应页面并关闭菜单

5. 点击菜单外部区域可以关闭菜单

优化细节：

1. 使用height替代max-height，避免收起时的卡顿感

2. 使用cubic-bezier(0.4, 0, 0.2, 1)缓动函数，使动画更加自然

3. 为下拉菜单项添加了渐入动画，提升用户体验

4. 优化了事件处理，防止事件冒泡导致的意外行为
